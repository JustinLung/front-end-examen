  • Adobe Photoshop

Dit is TPR!

Met TPR ga je leren hoe je een Java-Applicatie kan bouwen. Je kan met Java ook apps gaan bouwen! Hier in dit vak ga je vaak de volgende programma's gebruiken:

  • Je text editor (zoals: Atom of Visual Studio Code)

In deze periode ga je hiervoor 4 uur les krijgen in de week!

Dit is PRO!

PRO staat voor 'Programmeren'. In programmeren ga je leren hoe je met front-end en back-end gaat programmeren. Dit houdt in de voorkant van de website en de achterkant. Hieronder zie je de tools die ervoor nodig gaat hebben!

  • Atom (Of een andere text-editor naar jouw keuze)
  • Browser
  • XAMPP of MAMP
  • Filezilla
In deze periode ga je 4 uur les hiervan krijgen per week!

Dit is CMS!

CMS staat voor 'Content Management System'. In CMS Ga je leren hoe je met Wordpress gaat werken. Met Wordpress kan je makkelijk een website bouwen met behulp van drag & drop, maar met wordpress kan je natuurlijk ook een eigen thema of plugin gaan bouwen. Dit doe je met behulp van PHP. Hieronder zie je wat je nodig hebt:

  • Atom (Of een andere text-editor naar jouw keuze)
  • Filezilla
  • Wordpress
In deze periode ga je hiervan 2 uur les krijgen per week!

  • Atom (Of een andere text-editor naar jouw keuze)

  • MAMP of XAMPP

Dit is PROJ!

PROJ staat voor 'Project'. In PROJ Ga je het volgende doen. Je gaat opdrachten krijgen van je docenten. In die opdrachten kan het zo zijn dat je een website moet bouwen met verschillende functies daarin. In dit vak is het dus de bedoeling dat wat jij hebt geleerd in de voorgaande vakken, dat je die skills toepast in je project. Je gaat bij dit vak bijvoorbeeld een opdracht krijgen om een zelf gemaakte social media platform bouwen, of een soort memory card game. Je gaat de volgende programma's vaak gebruiken in dit vak:

  • Adobe Photoshop
  • Microsoft Word
  • Microsoft Powerpoint
  • Microsoft Excel
  • Atom (Of een andere text-editor naar jouw keuze)
  • Filezilla
  • MAMP of XAMPP
Dit is eigenlijk dus vooral een 'doe-vak'. Dit vak krijg je 2 uur per week in deze periode!
